# ListingCreatedEventHandler

`ListingCreatedEventHandler` is an asynchronous event handler responsible for processing domain events related to the creation of listings within the marketplace system. It encapsulates business logic triggered when a new listing is created, ensuring associated workflows such as notifications, indexing, or auditing are executed in response.

## API

### `ListingCreatedEventHandler`
Initializes a new instance of the `ListingCreatedEventHandler` class.
