# SellerComparisonDashboardDto

A data transfer object used to compare a seller's performance metrics against marketplace averages in the `marketplace-engine` project. It provides a consolidated view of key seller statistics, including revenue, listings, and ratings, along with comparative indicators to highlight performance relative to the marketplace.

## API

### `SellerId`
A unique identifier for the seller. Used to reference the seller in the system.

### `SellerName`
The display name of the seller. Provides a human-readable identifier for the seller.

### `ActiveListings`
The number of active listings the seller currently has available. This is a direct count of active inventory.

### `TotalRevenue`
The total revenue generated by the seller across all transactions. Expressed in the system's currency.

### `AverageRating`
The average rating of the seller, calculated from buyer feedback. Expressed as a double value (e.g., 4.5).

### `MarketplaceAverageRevenue`
The average revenue generated by all sellers in the marketplace. Used as a benchmark for comparison.

### `MarketplaceAverageListings`
The average number of active listings across all sellers in the marketplace. Serves as a baseline for inventory comparison.

### `MarketplaceAverageRating`
The average seller rating across the entire marketplace. Used to evaluate relative performance in ratings.

### `RevenueAboveAverage`
The difference between `TotalRevenue` and `MarketplaceAverageRevenue`. Indicates how much more (or less) revenue the seller generates compared to the average.

### `ListingsAboveAverage`
The difference between `ActiveListings` and `MarketplaceAverageListings`. Shows how the seller's inventory compares to the marketplace average.

### `RatingDifference`
The difference between `AverageRating` and `MarketplaceAverageRating`. Reflects the seller's rating advantage (or disadvantage) relative to the marketplace.

### `IsRevenueAboveAverage`
A boolean indicating whether the seller's `TotalRevenue` exceeds the `MarketplaceAverageRevenue`. `true` if above average, `false` otherwise.

### `IsListingsAboveAverage`
A boolean indicating whether the seller's `ActiveListings` exceed the `MarketplaceAverageListings`. `true` if above average, `false` otherwise.

### `IsRatingAboveAverage`
A boolean indicating whether the seller's `AverageRating` exceeds the `MarketplaceAverageRating`. `true` if above average, `false` otherwise.
