# ListingCreatedEventHandlerExtensions

`ListingCreatedEventHandlerExtensions` provides a set of utility methods for handling `ListingCreatedEvent` events with common cross-cutting concerns such as retries, timeouts, logging, and error handling. These extensions are designed to standardize event processing workflows in the `marketplace-engine` project, ensuring consistency in error handling and operational resilience.
