# Security Policy

## Supported Versions
- 1.x

## Reporting a Vulnerability

We take the security of Marketplace Engine seriously. If you discover a security vulnerability, please do not disclose it publicly.

Instead, please use one of the following methods to report the vulnerability:
- Use GitHub's built-in private advisory feature.
- Contact rutova2@gmail.com directly.

You can expect an acknowledgment within 48 hours of your report.
